THIS IS A REFERENCE FILE FOR THE NEW INVENTORY SECTION IMPLEMENTATION

The new inventory section will include:

1. Product Form with all new fields:
   - Product Status (Active/Inactive) dropdown
   - Main Image upload with preview
   - Additional Images (up to 6) with thumbnails
   - Prescription Required checkbox  
   - Category dropdown (Tablets/Syrup/Drops/Ointment/Injection/Supplement/Device)
   - Sub-category dropdown (Fever/Cold & Cough/Diabetes/BP/Skin/Pain Relief/etc)
   - How to Use textarea
   - Dosage Information textarea
   - Safety Information textarea

2. Product List with enhanced cards:
   - Show main image thumbnail
   - Show prescription required badge
   - Show product status (Active/Inactive) badge
   - Edit button (opens form in edit mode with pre-filled data)
   - Delete button (with confirmation)
   - Stock In button (opens modal)
   - Stock Out button (opens modal)
   - View Details button (shows all info including images)

3. Stock Management Modal:
   - Stock In/Out selection
   - Quantity input
   - Reason textarea
   - Date auto-filled
   - Stock history display

The implementation will be done directly in the ChemistDashboard.jsx file.
